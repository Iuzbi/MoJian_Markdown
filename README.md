# MoJian Markdown

> 一款面向 Windows 的 Markdown 阅读与编辑桌面应用，基于 Electron + React 构建，提供 `.exe` 安装器、文件关联、拖拽打开、双栏编辑预览与多套护眼主题。

## 项目简介

MoJian Markdown 不是网页，而是一款真正可以安装到 Windows 上的 Markdown 阅读与编辑器。

当前仓库：

- GitHub：[Iuzbi/MoJian_Markdown](https://github.com/Iuzbi/MoJian_Markdown)

当前版本重点围绕“更像桌面软件”的使用逻辑展开：

- 顶部操作收进真实窗口标题栏
- 文件、编辑、视图、帮助四个菜单重新组织
- 编辑模式与预览模式分工更清晰
- 左侧工作区缩窄，只保留必要信息
- 支持安装器分发、文件关联、拖拽打开和单实例接管

## 2.0.1 正式版亮点

### 标题栏重构

- 使用 Electron 自定义标题栏方案，将菜单直接放入窗口标题栏
- 顶部菜单顺序固定为：`文件 / 编辑 / 视图 / 帮助`
- 增加撤销、返回、左侧栏开关等常用操作

### 工作区优化

- 双栏编辑模式聚焦写作与预览
- 新增更纯粹的独立预览模式
- 左侧栏缩窄，只保留状态、模式、统计与最近文件
- 空白首页重新排版，更简洁、更稳定

### 交互体验

- 打开新文档前会检测未保存内容
- 支持拖拽文档到程序窗口
- 支持通过文档双击或快捷方式唤起当前实例
- 主题切换收纳在 `视图` 菜单中

## 快捷键

- `Ctrl + N`：新建文档
- `Ctrl + O`：打开文档
- `Ctrl + S`：保存文档
- `Ctrl + Shift + S`：另存为
- `Ctrl + Z`：撤销
- `Ctrl + Y`：返回
- `Ctrl + P`：切换到预览模式
- `Esc`：关闭菜单或帮助面板

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Electron | Windows 桌面壳层 |
| 前端界面 | React 19 | 主界面渲染 |
| 构建工具 | Vite | 前端开发与打包 |
| Markdown 渲染 | react-markdown + remark-gfm | 预览渲染 |
| 安装器 | electron-builder + NSIS | 生成安装器 |
