## 软件简介

MoJian Markdown 是一款专为 Windows 桌面环境设计的 Markdown 软件。

它不是网页，也不是浏览器插件，而是一个可以直接安装、直接运行的 `.exe` 桌面程序。整个产品围绕三个重点展开：

- 简洁清晰的桌面界面
- 稳定直接的本地文件编辑体验
- 尽量贴近日常写作与阅读习惯的交互逻辑

仓库地址：

- GitHub：[Iuzbi/MoJian_Markdown](https://github.com/Iuzbi/MoJian_Markdown)

## 核心特点

| 功能 | 说明 |
|------|------|
| Windows 安装器 | 提供 `.exe` 安装器，可直接安装到本地 |
| 文件关联 | 支持 `.md`、`.markdown`、`.txt` 文件双击打开 |
| 单实例接管 | 双击文件、拖入文件时，统一交给当前窗口打开 |
| 双栏编辑预览 | 左侧编辑、右侧预览，适合边写边看 |
| 预览模式 | 支持切换到单独预览界面，适合专注阅读 |
| 分栏可调 | 编辑区与预览区宽度可自由拖动调整 |
| 最近打开 | 自动记录最近使用的文档，方便继续处理 |
| 多主题 | 内置多套护眼、清爽的背景主题 |
| 简体中文界面 | 菜单、状态、帮助信息均为中文 |
| 按需日志 | 平时不常驻记录，仅在帮助菜单中手动开启 |

## 界面与使用体验

MoJian Markdown 当前版本采用更收敛的桌面化布局：

- 顶部为标题栏式菜单，包含 `文件`、`编辑`、`帮助`、`视图`
- 新建、打开、保存、另存为等操作统一放在 `文件`
- 标题、列表、引用、代码块、粗体等插入能力统一放在 `编辑`
- 主题切换与预览模式统一放在 `视图`
- 帮助菜单中可查看快捷键，也可以在需要时手动抓取日志

如果当前没有打开文档，软件会显示简洁欢迎页；
如果已经打开文档，则默认进入双栏编辑预览模式。

## 支持的打开方式

- 在软件内点击“打开文档”
- 直接双击 `.md`、`.markdown`、`.txt` 文件
- 将 Markdown 文件拖拽到程序窗口
- 将 Markdown 文件拖拽到软件快捷方式

如果软件已经在运行，新文件会直接在当前窗口中打开，而不会重复弹出多个实例。

## 快捷键

- `Ctrl + N`：新建文档
- `Ctrl + O`：打开文档
- `Ctrl + S`：保存文档
- `Ctrl + Shift + S`：另存为
- `Ctrl + Z`：撤销
- `Ctrl + Y`：恢复
- `Ctrl + P`：切换预览模式
- `Esc`：关闭菜单或从预览返回编辑

## 安装与运行

### 安装方式

在 GitHub Release 页面下载对应版本的安装器：

- `MoJian Markdown Setup x.x.x.exe`

运行安装器后，可以：

- 自定义安装位置
- 直接安装到某个盘符下新建的文件夹
- 自动创建桌面快捷方式

### 安装后产物

安装完成后，主程序为：

- `MoJian Markdown.exe`

用户可以通过桌面快捷方式、开始菜单或文件关联方式启动软件。

## 日志与问题排查

从 `v2.1.1` 开始，日志功能默认不常驻。

只有在需要排查问题时，才需要在 `帮助` 菜单中手动开启日志抓取。

日志目录位于安装目录下的 `logs` 文件夹。例如：

```text
F:\MoJian Markdown\logs
```

帮助菜单中支持：

- 开始抓取日志
- 停止抓取日志
- 打开日志目录

这样可以减少平时运行时的干扰，也方便在出现异常时快速定位问题。

## 适合谁使用

MoJian Markdown 适合以下使用场景：

- 需要在 Windows 本地长期写 Markdown 的用户
- 希望使用安装版软件，而不是网页工具的用户
- 习惯左写右看的双栏工作方式的用户
- 想要界面尽量简洁、安静、不复杂的用户
