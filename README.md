# MoJian Markdown

> 一款面向 Windows 的 Markdown 阅读与编辑桌面软件，基于 Electron + React 构建，提供 `.exe` 运行与安装分发能力。

## 项目简介

MoJian Markdown 是一个不是网页、而是可直接运行在 Windows 桌面的 Markdown 编辑器。

当前仓库：

- GitHub：[Iuzbi/MoJian_Markdown](https://github.com/Iuzbi/MoJian_Markdown)

这个项目主要强调三件事：

- 桌面软件而不是浏览器网页
- 简约、清晰、稳定的编辑体验
- 支持安装器、文件关联、拖拽打开、单实例接管

## 核心特性

| 模块 | 说明 |
|------|------|
| Windows 桌面应用 | 产物为 `.exe`，支持打包安装器 |
| 文件关联 | 支持 `.md`、`.markdown`、`.txt` 双击打开 |
| 拖拽打开 | 支持拖拽文件到程序窗口或快捷方式 |
| 单实例接管 | 重复打开文件时交给当前窗口处理 |
| 双栏编辑预览 | 左侧编辑、右侧预览，联动滚动 |
| 全屏预览模式 | 独立预览 Markdown 内容 |
| 可调分栏 | 编辑区和预览区宽度可拖动调整 |
| 最近打开 | 记录最近访问文档，方便继续处理 |
| 多主题 | 内置多套护眼、清爽的背景主题 |
| 简体中文界面 | 整体交互和文案统一为中文 |

## 2.0.1 正式版重点

### 标题栏重构

- 顶部改为真正的窗口标题栏布局
- 菜单顺序固定为：`左侧栏开关 / 文件 / 编辑 / 帮助 / 视图 / 撤销`
- 右上角只保留最小化、最大化、关闭三个窗口控件

### 菜单逻辑整理

- `文件`：新建、打开、保存、另存为、定位文件、进入预览、关闭文档
- `编辑`：撤销、恢复、H2、H3、列表、引用、代码块、粗体
- `视图`：左侧栏开关、预览模式、主题切换
- `帮助`：关于 MoJian Markdown、快捷键说明、功能说明

### 工作区优化

- 双栏模式下编辑区和预览区更聚焦
- 关闭最后一个文档后，回到简洁欢迎页而不是强制保留文档
- 左侧栏缩窄，只保留必要信息
- 首页改为更简约、更稳定的桌面软件布局

## 快捷键

- `Ctrl + N`：新建文档
- `Ctrl + O`：打开文档
- `Ctrl + S`：保存文档
- `Ctrl + Shift + S`：另存为
- `Ctrl + Z`：撤销
- `Ctrl + Y`：恢复
- `Ctrl + P`：切换预览模式
- `Esc`：关闭菜单、关闭帮助或退出全屏预览

## 技术栈

| 层级 | 技术 |
|------|------|
| 桌面框架 | Electron |
| 前端界面 | React 19 |
| 构建工具 | Vite |
| Markdown 预览 | react-markdown + remark-gfm |
| 安装器 | electron-builder + NSIS |

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建后的 Electron 运行产物默认位于：

```text
release/win-unpacked/
```

## 安装器

```bash
npm run dist
```

说明：

- 当前项目使用 `electron-builder` 生成 NSIS 安装器
- 如果网络环境限制了 NSIS 下载，安装器打包会失败，但 `win-unpacked` 仍可生成
